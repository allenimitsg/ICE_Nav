# Polar Ship Ice-Theta* Path Planning System
An intelligent path planning software system for polar ships navigating in fragmented ice regions, based on high-resolution optical imagery and improved Ice-Theta* algorithm.

## Overview
This project implements a novel path planning method for polar ships navigating in local fragmented ice areas. It solves the problems of low environmental perception accuracy, unsmooth paths generated by traditional grid-based algorithms, and the neglect of ice resistance effects.

The system integrates high-resolution optical remote sensing image processing and an improved Ice-Theta* algorithm to build a high-fidelity navigation environment and generate safe, smooth, and energy-efficient ship paths.

## Core Functions
1. **High-Fidelity Ice Environment Modeling**
   Based on high-resolution optical remote sensing images, the system uses an intensity-based segmentation algorithm to construct a multi-level navigable simulation environment, including open water, fragmented ice, and thick ice.

2. **Improved Ice-Theta* Path Planning**
   The proposed Ice-Theta* algorithm supports any-angle path planning. It uses an improved line-of-sight inspection mechanism and soft cost field integration to generate smooth paths that conform to ship navigation habits.

3. **Multi-Objective Cost Optimization**
   A multi-objective cost function is constructed, which considers path length, ice navigation resistance, and collision risk, guiding ships to select low-resistance and safe routes.

4. **Simulation and Performance Evaluation**
   Supports tests in random ice fields with different concentrations. Path length, navigation time, collision energy, and comprehensive energy consumption are automatically evaluated.

## Experimental Advantages
Compared with the traditional A* algorithm:
- Average path length reduced by **8.3%**
- Collision energy reduced by **30.8%**
- Navigation time shortened by up to **12.8%** (in low-density ice conditions)
- Smoother paths with fewer turns
- Lower energy consumption and higher navigation safety
